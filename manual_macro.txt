MacroFire 3.2.9


Macro settings
-------------------------------------------------
  記録したボタン操作を繰り返し実行することを可能にします。

  Add macro
    新しいマクロスロットを追加します。

  Remove macro
    現在のマクロスロットを削除します。
    記録されているマクロもろとも削除されるので注意してください。

  Macro
    マクロのスロット番号です。
    スロットごとにマクロは保存されます。
    この項目にカーソルを合わせて△ボタンをおすと、
    このマクロに名前を付けることができます。


  Preference
  -----------------------------------------------
    Buttons to run/stop the macro
      ゲーム中にメニューを表示させずにマクロを実行/停止するボタン。
      マクロ実行中にこのボタンを押すと逆に停止させることができる。

     Number of loop
      マクロを何回連続実行するかを設定します。
      0で無限ループします。

  Control
  -----------------------------------------------
    Run
      Number of loopの回数実行します。

    Stop
      実行中のマクロを停止します。

    Edit
      マクロを編集します。
      操作や、マクロコマンドタイプについては後述のEdit macroの項目を参照。

    Clear
      記録しているマクロをクリアします。


  Realtime recording
  -----------------------------------------------
    Start
      マクロのリアルタイム記録を開始します。

    Append
      現在のマクロの追加する形でリアルタイム記録を開始します。

    Stop
      マクロの記録を終了します。

    Recording analog stick movement (ON/OFF)
      マクロの記録において、アナログスティックの動作を記録するかを切り替えます。
      マクロ記録中でも設定を変更可能。


  File
  -----------------------------------------------
    Load
      メモリースティックからマクロファイルをロードします。

    Save
      メモリースティックへ、現在保持しているマクロを保存します。
      ホットキーやループ回数も記録します。


Edit macro
-------------------------------------------------
  Start recordingで記録したメインメモリ上のマクロを編集します。
  一行一行が実行すべきコマンドで、種類は以下のものが存在します。

    Delay: 一時停止する。
           直前のボタン状態を指定時間維持する。
           単位はms(ミリ秒)で1秒は1,000ミリ秒と等価。
           最大値は999,999,999ms(277時間46分39.999秒)になる。

    Buttons press: ボタンを押下する。
                   直前のボタン状態に追加される形でボタンを押す。
                     Buttons press: ×
                     Buttons press: ○
                   となっていると、最終的に○と×の二つが押される状態になる。

    Buttons release: ボタンを放す。
                       Buttons press同様に、直前の状態から指定されたボタンが放される形になる。
                       Buttons press  : ○ △
                       Buttons press  : ×
                       Buttons release: △
                     となっていると、最終的に○と×の二つが押される状態になる。

    Buttons change: ボタンの状態を変更する。
                    Buttons pressとButtons releaseが同時に起きたときに発生するコマンドで、
                    直前のボタン状態に関わらずボタン状態を完全に変更する。
                      Buttons press  : ○ △
                      Buttons press  : ×
                      Buttons release: △
                      Buttons change : □
                    となっていると、最終的に□ボタンのみが押される状態になる。

    Analog move: アナログスティックを傾ける。
                 左上が(x,y)=(0,0)として、アナログスティックを任意の位置へ移動させる。

    Rapidfire start: 連射を開始する。
                     Buttons pressと同じように連射を行うボタンを設定する。
                     PD/RDはPress delay/Release delayのことで、詳細はmanual_rapidfire.txtを参照。

    Rapidfire stop: 連射を停止する。


  エディットではこれらコマンドの値の変更と、
  これらコマンドそのものを挿入/削除を行うことができます。

  Delay以外の全てのコマンドは、1フレームに一つの実行されます。
  例えば、
      Buttons press: ○ ×
  は、完全に同時にボタンが押されますが、
      Buttons press: ○
      Buttons press: ×
  は、1フレームずれてボタンが順番に入力されます。
  従って、
      Buttons press: ○
      Buttons release: ○
  というマクロを無限に繰り返すと○ボタンの連射のような動作になります。

  ただしDelayだけは別で
      Delay: 500ms
  と
      Delay: 200ms
      Delay: 300ms
  はどちらも全く同じ意味で、同じ時間だけ待ちます。


  また、マクロを全てDeleteで削除しようとしても、完全には消せません。
  具体的には必ず Delay 0 ms が残ります。
  マクロを完全に消去するには、マクロのメインメニューからClearを実行してください。
